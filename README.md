# Task Tracker Web Application

A modern and responsive Task Tracker web application built using React and Vite, designed to help users efficiently organize, manage, and track daily tasks through a clean and interactive user interface.

## Live Demo

https://maheshchilukamari.github.io/task-tracker/

---

## Project Overview

This project demonstrates practical frontend development skills, component-based architecture, production deployment workflows, and modern web application practices. The application was developed using React for dynamic UI rendering and Vite for optimized development and production performance.

The project also includes automated deployment using GitHub Actions and GitHub Pages, showcasing CI/CD integration and cloud-based hosting workflows.

## Installation & Setup

Clone the repository:

git clone https://github.com/maheshchilukamari/task-tracker.git

Navigate into the project folder:

cd task-tracker

Install dependencies:

npm install

Run the development server:

npm run dev

Build for production:

npm run build

Deploy application:

npm run deploy

---

## Deployment

The application is deployed using GitHub Pages with automated CI/CD workflows configured through GitHub Actions.
